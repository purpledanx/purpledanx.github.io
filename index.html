About Me

I’m purpledan. I code Roblox games and occasionally build websites. I value perfection and excellence in my work, but I also recognize that I don’t know everything - and I embrace every learning opportunity to its fullest. I started coding when my school put a blocking extension on my Chromebook. I got frustrated that I couldn’t access CoolMathGames in class anymore, so I learned website building with GitHub to create unblocked sites for my classmates. At the same time, Roblox was my favorite game, and I realized I could try making better games then the ones I was playing. That’s when I started coding Roblox games. Four years later, I’m still coding and always looking for ways to improve.
